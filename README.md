# Lux Python Demo

Ce projet est une démonstration de l'utilisation de la bibliothèque Lux en Python pour l'exploration de données.

## Prérequis

- Python 3.6 ou supérieur
- pip

## Installation


- pip install lux-api
- pip install lux-widget
- pip install seaborn
